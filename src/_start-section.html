Начните пользоваться сервисом за 3 простых шага

  • Лид - магнит

    Сделайте полезный материал для аккаунта: мануал, чек-лист или видеоурок. Загрузите его в облако или на Ютуб с доступом только по ссылке.

  • Регистрация

    Создайте в Client-Turbine «подписную страницу» с описанием ценности вашего материала. Укажите логин аккаунта, куда должен подписаться пользователь для получения материала. Пароль не требуется.

  • Запуск рекламы

    Сервис сгенерирует ссылку, по которой после подписки можно получить материал. Запустите таргетированную рекламу в Инстаграме с призывом бесплатно получить ваш материал. Прикрепите к рекламе ссылку.
